## 📋 Google Search Console Kurulumu

### Adım 1: Google Search Console'a Kayıt
1. https://search.google.com/search-console/ adresine gidin
2. "Özellik Ekle" butonuna tıklayın
3. "URL öneki" seçeneğini kullanın: `https://purivita.com.tr`
4. Doğrulama yöntemlerinden birini seçin:
   - HTML dosyası (önerilen)
   - HTML etiketi
   - Google Analytics
   - Google Tag Manager
   - Domain name provider

### Adım 2: Sitemap Gönderimi
1. Google Search Console'da sol menüden "Sitemap'ler" seçin
2. "Yeni sitemap ekle" kutusuna şunu yazın: `sitemap.xml`
3. "Gönder" butonuna tıklayın

### Adım 3: URL İnceleme
1. Ana sayfanızı test edin: `https://purivita.com.tr`
2. "URL İNCELEME" aracını kullanın
3. "DİZİNE ALMA İSTEYİN" butonuna tıklayın

## 📊 Google Analytics Kurulumu

### Adım 1: Google Analytics Hesabı Oluşturma
1. https://analytics.google.com/ adresine gidin
2. "Ölçüme Başla" butonuna tıklayın
3. Hesap ve özellik bilgilerini girin

### Adım 2: Tracking Code Ekleme
Google Analytics'ten aldığınız tracking code'u `index.html` dosyasının `<head>` kısmına ekleyin:

```html
<!-- Google Analytics -->
<script async src="https://www.googletagmanager.com/gtag/js?id=G-XXXXXXXXXX"></script>
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('js', new Date());
  gtag('config', 'G-XXXXXXXXXX');
</script>
```
